# contact_book
A simple command-line contact management system built in Python.

## Features
- Add new contacts (name, phone, email)
- Search for a contact by name
- Delete a contact by name
- View all saved contacts

## How it works
Contacts are stored as a list of dictionaries in memory. The program 
runs on a menu loop that lets the user choose an action until they exit.

## Skills demonstrated
- Functions with parameters
- Lists and dictionaries
- Loops (for/while)
- Conditional logic

## How to run
python contact_book.py
